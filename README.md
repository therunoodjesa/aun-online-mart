# 🛒 AUN Online Mart

AUN Online Mart (AOM) is a mobile marketplace application designed for the American University of Nigeria community. It enables students, staff, and local vendors to buy, sell, and order products and food from a single platform.

## ✨ Features

- 👤 User authentication
- 🛍️ Multi-vendor marketplace
- 🍔 Food ordering
- 🚚 Delivery request management
- 🔍 Product search and filtering
- ❤️ Wishlist
- 🛒 Shopping cart
- 💳 Secure checkout
- 📦 Order tracking
- 👤 User profiles

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/therunoodjesa/aun-online-mart.git
```

### Navigate into the project

```bash
cd aun-online-mart
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npx expo start
```
